Notes on the created application:

  • Returns the usd bitcoin price.
  • Shows the opening price for each day.
  • All prices are rounded down to a full dollar value.
  • When checking for prime numbers in the price, uses only single digit prime numbers.
  • A start date may be filled in without an end date. The current date will be used as the end date.
  • The current day's info cannot be displayed as its only available on the next day.


Technologies used:

  • Angular1/JS
  • Angular Material Design.
  • npm package manager.
  • bower for additional package management.
  • grunt to run the application.
  • yo generator to set up the application.
  • BITFINEX rest webservice to retrieve dynamic info.
  • GIT for version management and sharing the application.
